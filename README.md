# 📊 Coursera Microsoft Excel Project

## 📌 Project Overview

This project was completed as part of the **Coursera Microsoft Excel Course**. It demonstrates the application of Microsoft Excel for data analysis, reporting, and dashboard creation using real-world business scenarios. The project focuses on transforming raw data into meaningful insights through interactive reports and visualizations.

---

## 🚀 Features

- Data Cleaning and Preparation
- Advanced Excel Formulas
- Interactive Pivot Tables
- Dynamic Pivot Charts
- Dashboard Creation
- Slicers for Interactive Filtering
- Data Visualization
- Conditional Formatting
- Data Validation
- Business Reporting
